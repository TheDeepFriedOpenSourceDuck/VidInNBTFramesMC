## Installation

### Linux:

Clone the Repository from Github:
```bash
git clone https://github.com/TheDeepFriedOpenSourceDuck/VidInNBTFramesMC.git
```
Go into the directory:
```bash
cd VidInNBTFramesMC
```
make it:
```bash
make
```
